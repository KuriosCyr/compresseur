# Guide d'utilisation des animations 360° - Option Autonome

Bienvenue dans le dossier de votre animation en 360° générée avec la version démo de 360PIK. Cette version autonome est spécialement conçue pour être utilisée hors ligne, idéale pour les représentants et commerciaux qui souhaitent présenter l'animation sans dépendre d'une connexion Internet.

## À propos de l'Option Autonome

L'option Autonome vous permet de visualiser votre animation sans avoir besoin d'une connexion Internet. Toutes les ressources nécessaires sont incluses dans ce dossier. Il est crucial de ne pas supprimer le dossier "assets" inclus, car il contient les dépendances essentielles à la bonne marche de l'animation.

## Utilisation de l'animation

### Sur un ordinateur :

1. Extraction du fichier ZIP : Si votre animation a été fournie sous forme de fichier ZIP, commencez par l'extraire dans un dossier de votre choix.
2. Lancement de l'animation : Ouvrez le dossier extrait et double-cliquez sur le fichier HTML pour lancer l'animation dans votre navigateur par défaut.

### Sur une tablette ou un smartphone :

1. Transfert du fichier : Transférez le dossier extrait sur votre tablette ou smartphone via un câble USB ou toute autre méthode de votre choix.
2. Ouverture du fichier HTML : Utilisez un gestionnaire de fichiers sur votre appareil pour naviguer jusqu'au dossier où vous avez transféré l'animation. Touchez le fichier HTML pour l'ouvrir avec le navigateur par défaut de votre appareil.
